# Contributing to the VIT Network Organization

This repository provides the **default** community health files (Code of Conduct, Security Policy,
issue/PR templates) inherited by every repository in the Value-intelligence-trust organization that does not define
its own.

## General Contribution Workflow (applies org-wide)

1. Fork the target repository and branch from `main`.
2. Keep changes focused and well-tested.
3. Follow [Conventional Commits](https://www.conventionalcommits.org/).
4. Open a PR using the standard template and request review.
5. All contributors must follow the [Code of Conduct](./CODE_OF_CONDUCT.md).

For repository-specific contribution guidelines, see that repository's own `CONTRIBUTING.md`.
